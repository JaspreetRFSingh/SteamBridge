# StreamBridge Project Summary

## Overview
StreamBridge is a production-grade CDC pipeline built on Apache Kafka Connect.

## Features
- Custom REST source connector
- Debezium PostgreSQL CDC integration
- PII masking and topic routing transforms
- Elasticsearch sink connector
- Docker Compose local development environment
- Kubernetes Helm charts for production
- Prometheus and Grafana monitoring

## Technology Stack
- Java 17, Maven
- Apache Kafka 3.x
- Debezium 2.5.0
- PostgreSQL 15
- Elasticsearch 8.11
- Docker, Kubernetes, Helm
